# Riego Extractor v1.5

## Instalación de dependencias

```bash
pip install -r requirements.txt
```

## Generar ejecutable .exe

```bash
python build.py
```

El ejecutable se generará en: `dist/Riego Extractor v1.5.exe`

## Uso

Ejecutar `Riego Extractor v1.5.exe` en cualquier PC con Windows (sin necesidad de Python).
